## 实现功能

* 新增 - 两种形式

1. 输入 + 新增按钮
2. 输入 + 回车

* 修改 - 点击选项进行修改，点击其他地方完成修改，或者点击【完成】按钮

* 删除 - 点击删除按钮完成删除
